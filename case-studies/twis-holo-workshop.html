Case study

Twis Holo Workshop

Local-first AI creative workspace and remote field kit deployed through Cloudflare Pages.

Live project proof: ollie-twis-holo-workshop.pages.dev

Problem

I needed a browser-accessible AI creative workshop without turning private local files, source authority, Canon decisions, or personal folders into cloud property.

What I built

  • Browser workspace with rooms for Talk, Write, Music, Images, Video, My Work, Modules, and Settings.
  • Cloudflare Pages public remote hull.
  • Workers AI away-mode endpoint.
  • Browser speech recognition and speech synthesis voice loop.
  • Local brain setup path for localhost model use.
  • Setup documentation and static contract checks.

My role

Project builder, AI workflow designer, practical tester, deployment troubleshooter.

Boundary preserved

Cloudflare is the remote field kit. The local PC remains the private source authority for folders, Canon, and private artifacts.
